## 附錄 B ─ 常數字元對照

| 常數 | 數值 | 意義 |
|------|------|------|
| `STDIN_FILENO` | 0 | 標準輸入 |
| `STDOUT_FILENO` | 1 | 標準輸出 |
| `STDERR_FILENO` | 2 | 標準錯誤 |
| `O_RDONLY` | 0 | 唯讀 |
| `O_WRONLY` | 1 | 唯寫 |
| `O_RDWR` | 2 | 讀寫 |
| `O_CREAT` | 64 | 不存在就建立 |
| `O_TRUNC` | 512 | 存在就清空 |
| `O_APPEND` | 1024 | 附加在檔案尾端 |
