## 前言

當你寫過一般應用程式後，總有一天會好奇：「程式是怎麼和作業系統打交道的？」
答案就是**系統呼叫（system call）**。

本書會帶你認識 Unix/Linux 中最核心的八個系統呼叫：

| 系統呼叫  | 用途 |
|-----------|------|
| `open`    | 開啟檔案 |
| `close`   | 關閉檔案 |
| `read`    | 從檔案讀取資料 |
| `write`   | 寫入資料到檔案 |
| `fork`    | 建立新行程（process） |
| `execvp`  | 載入並執行另一個程式 |
| `dup2`    | 複製檔案描述子（redirect） |

以及三個標準的檔案描述子（file descriptor）：

| 編號 | 名稱 | 用途 |
|------|------|------|
| 0    | stdin  | 標準輸入（通常是鍵盤） |
| 1    | stdout | 標準輸出（通常是螢幕） |
| 2    | stderr | 標準錯誤（通常是螢幕） |

所有範例都用 **C 語言**撰寫，並在 Linux 上編譯執行。
建議你親手打字、編譯、修改，才能真正學會。
